# 🤖 Telegram Бот-Консультант — Инструкция по запуску

## Что нужно для запуска

1. Python 3.10+
2. Токен бота от @BotFather
3. Claude API ключ (console.anthropic.com)
4. ID твоего Telegram канала

---

## Шаг 1 — Создай бота в Telegram

1. Напиши @BotFather → /newbot
2. Задай имя и username боту
3. Скопируй токен

---

## Шаг 2 — Добавь бота в канал как АДМИНИСТРАТОРА

⚠️ ВАЖНО: бот должен быть администратором канала!
Только так он будет получать события о новых подписчиках.

Права администратора которые нужны:
- ✅ Добавление участников (обязательно)

---

## Шаг 3 — Узнай ID канала

Перешли любое сообщение из канала боту @username_to_id_bot
Или используй @getidsbot

ID будет выглядеть так: -1001234567890

---

## Шаг 4 — Настрой переменные окружения

```bash
cp .env.example .env
nano .env  # или открой любым редактором
```

Заполни:
- BOT_TOKEN — токен от BotFather
- CLAUDE_API_KEY — ключ с console.anthropic.com
- CHANNEL_ID — ID канала (с минусом!)
- ADMIN_USERNAME — твой username без @

---

## Шаг 5 — Заполни тарифы в bot.py

Открой bot.py, найди раздел SYSTEM_PROMPT.
Замени [Вставь тарифы и цены] на актуальные данные с neirosetka.ru.

---

## Шаг 6 — Установи зависимости и запусти

```bash
pip install -r requirements.txt
python bot.py
```

---

## Деплой на Railway (рекомендую)

1. Зарегистрируйся на railway.app
2. Создай новый проект → Deploy from GitHub
3. Загрузи папку на GitHub
4. В настройках Railway добавь переменные окружения (те же что в .env)
5. Railway сам запустит бота

Стоимость: ~$5/мес или бесплатно (500 часов/мес)

---

## Деплой на Render (альтернатива, есть бесплатный план)

1. render.com → New → Background Worker
2. Укажи команду: python bot.py
3. Добавь переменные окружения

---

## Частые проблемы

❓ Бот не реагирует на новых подписчиков
→ Проверь что бот — администратор канала

❓ "Не удалось отправить сообщение"  
→ Пользователь запретил сообщения от ботов (Privacy settings)
→ Это нормально, бот пропускает таких пользователей

❓ Claude API ошибка
→ Проверь что ключ правильный и есть баланс на аккаунте
